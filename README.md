# Hungarian name day API

**See this app live on the web (Hungarian):** https://nevnapok.vercel.app

## Getting Started

**Clone the repo:**

```sh
git clone git@github.com:jozsefsallai/nevnapok-api.git
cd nevnapok-api
```

**Install the dependencies:**

```sh
npm i -g yarn
yarn
```

**Run the dev server:**

```sh
yarn dev
```

**Make sure your changes pass linting:**

```sh
yarn lint
```
